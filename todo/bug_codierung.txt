Beschreibung:

Derzeit werden nur unterschiedliche Codierungen auf unterschiedlichen Systemen unterstützt. Auf Windows 7 wird z.B. nur die Windows Codierung korrekt dargestellt, während auf Windows 10 nur UTF-8 funktioniert. Besser wäre es das zu vereinheitlichen, sodass z.B. UTF-8 überall korrekt dargestellt wird, indem beim Importieren und Exportieren von Files die Codierung mitangegeben wird.
